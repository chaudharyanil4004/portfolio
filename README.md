# Portfolio Website

This is a simple, responsive portfolio template built with HTML, CSS and vanilla JavaScript.

Files included:
- index.html — main page
- styles.css — styling
- script.js — small interactive bits
- assets/* — put your images, profile photo, resume, favicon here

How to use
1. Replace placeholder text (Your Name, city, email) in index.html.
2. Replace images in the `assets` folder:
   - profile-placeholder.jpg
   - project1.jpg, project2.jpg, project3.jpg
   - favicon.svg (optional)
3. (Optional) Configure contact form:
   - The form action points to Formspree by default. Create a Formspree form and replace the action URL.
   - Or leave it and the "Email me" button will open mail client.

Preview locally
- Open `index.html` in your browser.

Deploy with GitHub Pages
1. Create a repository (or use this one).
2. Push these files to the repository's `main` branch (or `gh-pages` branch).
3. In GitHub Settings → Pages, set the source to the branch and the root folder. GitHub will publish at `https://<your-username>.github.io/<repo>/`.
